Välkommen till Jävla välgörenhet!

Vi på Jävla välgörenhet tycker att det är viktigt att alla människor behandlas lika och med respekt. För att uppmuntra detta gör vi allt vi kan för att hjälpa de som behöver och tar tid till att visa uppskattning för de som gör gott i världen.

För att göra världen ett så bra ställe som möjligt vill vi på Jävla välgörenhet att alla levande varelser behandlas med respekt och att alla gör så gått de kan för att minska lidande hos allt som lever.
